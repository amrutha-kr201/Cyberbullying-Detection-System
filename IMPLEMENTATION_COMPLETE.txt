🎨 USER EXPERIENCE
================================================================================

Login Flow:
  1. User visits site
  2. Redirected to login page
  3. Enters credentials
  4. Authenticated via Supabase
  5. Session created
  6. Redirected to main page

Sign Up Flow:
  1. User clicks "Sign Up"
  2. Fills registration form
  3. Account created in Supabase
  4. Redirected to login
  5. Logs in with new credentials

Detection Flow:
  1. User logged in
  2. Chooses detection method
  3. Submits text/image/dataset
  4. ML model analyzes
  5. Results displayed
  6. Saved to file & database

Logout Flow:
  1. User clicks "Logout"
  2. Session cleared
  3. Redirected to login
  4. Cannot access without re-login

Fix: Check Supabase dashboard for errors

Issue: Login fails
Fix: Verify email and password are correct

Issue: Logged out automatically
Fix: Normal on server restart, just login again

Issue: test_auth_setup.py shows errors
Fix: Read the error message and follow instructions
