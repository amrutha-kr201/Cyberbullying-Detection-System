STEP 1: Install Tesseract OCR (One-time setup)
-----------------------------------------------
1. Go to: https://github.com/UB-Mannheim/tesseract/wiki
2. Download: tesseract-ocr-w64-setup-5.x.x.exe
3. Run the installer
4. Use default installation path
5. Done!


STEP 2: Use Image Detection
----------------------------

METHOD A: Detect from Image Files
----------------------------------

   C:\Users\navee\Downloads\cyberbullying\

2. Run: python detect_from_image.py

3. The script will automatically:
   - Find all images (.jpg, .png, etc.)

   - Show results


METHOD B: Manual Text Input (No Tesseract needed)

1. Run: python detect_from_image.py
